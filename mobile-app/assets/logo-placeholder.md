# Logo File Required

Please add the Doc Time logo image as `logo.png` in this directory.

The logo should be:
- PNG format with transparency
- Shows the teal stethoscope and "Doc Time" text
- Recommended size: 400x240 pixels (or 2:1 aspect ratio)
- White/transparent background (will be displayed on #393D7E background)

Once added to `mobile-app/assets/logo.png`, it will automatically appear on the Sign Up and Login screens.
